# Paraná Agro

Site informativo sobre como será a agricultura do futuro no Paraná.

---

## Sobre

O Paraná Agro apresenta de forma simples os principais temas que vão transformar o campo paranaense — de novas tecnologias a desafios estruturais que precisam ser superados.

---

## Estrutura dos arquivos

```
parana-agro/
├── index.html   — estrutura da página
├── style.css    — estilização
└── main.js      — accordion
```

---

## Como usar

Baixe os três arquivos na mesma pasta e abra o `index.html` no navegador. Nenhuma instalação necessária.

---

## Conteúdo do site

- **O futuro** — como o campo paranaense vai produzir mais com menos recursos
- **O que vai mudar** — precisão, bioinsumos, solo vivo e venda direta
- **Desafios** — internet, capacitação, crédito, jovens e água
